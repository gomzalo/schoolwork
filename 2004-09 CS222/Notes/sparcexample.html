Commands and Instructions

  1. Any line that starts with an "!" is a comment. Put there so that you can read and understand the program. They are vital for you the programmer.
  2. The next 4 lines starting with ".section" never change and tells the computer that everything up to ".text" is an instruction section. It is a read only section. It also tells the computer to start excuting at "main".
  3. The next set of comments gives an identification to the listing. Up to this point can be put in a template.
  4. The next set of comment is a program description.
  5. The next 3 instructions prints the ID banner for the output.
  6. The next three instructions print out a prompt. Note register %o0 is used for both strout and decin.
  7. The next three instruction reads a value and puts it in register L0.
  8. The next 6 instructions prints out a prompt, reads a value and puts it in register L1.
  9. The next instruction adds the two numbers and puts the result in L2.
  10. The next 10 instructions prints a heading and the sum
  11. The next 3 instructions print out the Normal Termination meassage.
  12. The epilogue statements returns control to the system.
  13. The data section is used to set up memory. The ".section data" must be there exactly as is. It is part of the template is is the epilogue.
  14. ascii is like dc.b and asciiz is like dc.b with an automatic null terminator added.
