UNIX Commands

Some useful UNIX commands: (compiled by Adam Knibbs, 2 June 2003)

cd this is used to change directory.

cd directory will put you into the directory called directory

cd .. this will bring you back one level in your directory

man command this will allow you to look at the manual (man) for any command that you type in. If this does not bring up any information the command might be an alias for another one.

pwd this command will tell you where you are by printing working directory

ls this command will list all the files in short form.

ls -a this command will list all the files including hidden files.

ls -l this command will list all the information about your files, such as file size

ps this lists the process status that you are running (this is useful if you have a process that has stopped responding) Look for the number(called the process id or pid for short) that is beside the program that you have running. Using this with kill will stop the program.

kill number this is used to kill processes that are not responing. Used with the number that you get from ps

quota -v this will list how much of your space you are using. Important if you are getting blank files that should have data in them.
