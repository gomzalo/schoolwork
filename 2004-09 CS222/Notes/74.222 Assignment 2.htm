  1. (20 marks) Write a program that calculates and prints the numbers 1 to 9 and the cubes of those numbers in a table as shown below. 

        x     x^3
    ----------
    1     1
    2     8
    3     27
      ...
    9     729
  2. (20 marks) Write a program to read n, and then n integers. Count the number of positive, zero, and negative integers in the set of n integers, and print out your findings with titles and the counts. Prompt for the n, echo n, and echo the n integers. Handle n negative with an error, and handle n = 0 properly. Use the data sets A2Q2D1.txt , A2Q2D2.txt , A2Q2D3.txt , A2Q2D4.txt .

  3. (20 marks) Write a program to read a list of integers until the sentinel value -99999 is encountered; echo the input. Excluding the sentinel, calculate the number of integers, their sum, their minimum, and their maximum. Print the number of integers, the sum, their minimum, and their maximum. Find and print the average of the integers correctly rounded to the nearest integer. Give an error message if the first number is the sentinel. Use the data sets A2Q3D1.txt , A2Q3D2.txt , A2Q3D3.txt , A2Q3D4.txt , A2Q3D5.txt .

  4. (20 marks) Write a program to read in a string of characters (up to 80 characters) and perform the following conversions;
    • Convert all lowercase letters to uppercase.
    • Convert all digits to '*' (asterisk).
    • Convert all blanks to '_' (underscore).
    • Leave all other characters as is.
    Prompt for the string and echo the original string as well as the converted string. Use the input files A2Q4D1.txt , A2Q4D2.txt , A2Q4D3.txt , A2Q4D4.txt , A2Q4D5.txt .
